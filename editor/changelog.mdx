<Update label="v1.1.0" description="March 24, 2026">
### Overview
Squeditor v1.1.0 introduces significant improvements to the web editor, including new search functionality, drag-and-drop table creation, and enhanced API playground capabilities.

### Improvements
- New search functionality in the web editor makes it easier to find and navigate to files across your documentation.
- Drag to add table rows and columns directly in the editor for faster table creation and modification.
- The API playground now supports audio response examples, making it easier to document audio-related API endpoints.

### Bug fixes
- Fixed layout shift issues in the Sequoia theme.
- Fixed logout timeout issues.
- Fixed accordion ID deduplication to prevent duplicate anchor links.
- Improved SVG sanitization for better security.
</Update>
